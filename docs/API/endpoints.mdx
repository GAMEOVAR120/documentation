## Server

### Get Server

**Route**<br/>
`GET /api/v1/servers/:serverId`

**Description**<br/>
Return info of a specific server.

**Authentication**<br/>
`API Key`

**Path Parameters**<br/>
```json
// ServerID (required)
:serverId

// example 
bc730283-4033-43e3-a8f8-f511606bce2a
```

**URL Parameters**<br/>
```json
// Filter Parameter (optional)
// Additional filter to specify how much data you want to fetch.
filter: None, Minimal, Status
// None: Default, show all server info (serverId, status, name, description, pathToFolder, folderName, creationDate, isSetToAutoStart, keepOnline, javaAllocatedMemory, javaStartupLine)
// Minimal: Returns serverId, status, name, description & creationDate
// Status: Returns serverId & status

// example 
?filter=Minimal
```

### Run Server Action

**Route**<br/>
`POST /api/v1/servers/:serverId/execute/action`

**Description**<br/>
Change the status of a specific server.

**Authentication**<br/>
`API Key`

**Path Parameters**<br/>
```json
// ServerID (required)
:serverId: guid

// example 
bc730283-4033-43e3-a8f8-f511606bce2a
```

**Request Body**<br/>
```json
// Action (required)
action: integer
// 0 -> does nothing
// 1 or Stop
// 2 or Start
// 3 or Kill
// 4 or Restart

// example 
{
  "action" : 1
}
```

## Console

### Get Server Console

**Route**<br/>
`GET /api/v1/servers/:serverId/console`

**Description**<br/>
Return the server console.

**Authentication**<br/>
`API Key`

**Path Parameters**<br/>
```json
// ServerID (required)
:serverId

// example 
bc730283-4033-43e3-a8f8-f511606bce2a
```

**URL Parameters**<br/>
```json
// AmountOfLines Parameter (required)
// Amount of lines you want to fetch, -1 to fetch everything.
amountOfLines: integer

// TakeFromBeginning Parameter (optional)
// Specify if you want to start fetching from the oldest lines first.
takeFromBeginning: boolean

// Reversed Parameter (optional)
// Option to flip the order of the returned console lines.
reversed: boolean

// example 
?amountOfLines=50?takeFromBeginning=false?reversed=false
```


### Check if Console is outdated

**Route**<br/>
`GET /api/v1/servers/:serverId/console/outdated`

**Description**<br/>
Check if your version of the console is outdated. This by sending the last 2 last lines for comparison.

**Authentication**<br/>
`API Key`

**Path Parameters**<br/>
```json
// ServerID (required)
:serverId

// example 
bc730283-4033-43e3-a8f8-f511606bce2a
```

**URL Parameters**<br/>
```json
// SecondLastLine Parameter (required)
secondLastLine: string

// LastLine Parameter (required)
lastLine: string


// example 
// secondLastLine: [19:25:31] [Server thread/INFO]: Closing Thread Pool
// lastLine: [19:25:31] [Server thread/INFO]: Closing Server
// after URI Encoding the Parameters, it will look like:
?secondLastLine=%5B19%3A25%3A31%5D%20%5BServer%20thread%2FINFO%5D%3A%20Closing%20Thread%20Pool?lastLine=%5B19%3A25%3A31%5D%20%5BServer%20thread%2FINFO%5D%3A%20Closing%20Server
```

### Create Scheduler Task

**Route**<br/>
`POST /api/v1/servers/:serverId/scheduler/tasks`

**Description**<br/>
Create a new task.

**Authentication**<br/>
`API Key`

**Path Parameters**<br/>
```json
// ServerID (required)
:serverId

// example 
bc730283-4033-43e3-a8f8-f511606bce2a
```

**Request Body**<br/>
```json
// Name (required)
name: string

// Enabled (optional)
enabled: boolean

// Timing (required)
timing: object

// Job (required)
job: object
```
Timing
```json
// FixedTime
repeat: boolean
timeSpan: TimeSpan

// Interval
repeat: boolean
interval: integer

// Timeless
// no parameters
```
Job
```json
// Server Action
action : ServerAction
// 0 or Stop
// 1 or Start
// 2 or Kill
// 3 or Restart

// Run Commands
commands: string[]

// Start Backup
backupIdentifier: guid
```

```json
// example (Interval & Run Commands)
{
  "name": "Made with the API <3",
  "enabled": true,
  "timing": {
      "repeat": true,
      "interval": 150
  },
  "job": {
      "commands": [
        "/say hello from the api"
    ]
  }
}
```


### Edit Scheduler Task

**Route**<br/>
`PUT /api/v1/servers/:serverId/scheduler/tasks/:taskId`

**Description**<br/>
Update a specific server task.<br/>
You can not change the timing or job of the original task, only its values.

**Authentication**<br/>
`API Key`

**Path Parameters**<br/>
```json
// ServerID (required)
:serverId
// example 
bc730283-4033-43e3-a8f8-f511606bce2a

// TaskId (required)
:taskId
// example 
8ce91a2f-5b99-486b-bf3d-af823a999824
```

**Request Body**<br/>
```json
// Name (optional)
name: string

// Enabled (optional)
enabled: boolean

// Timing (optional)
timing: object

// Job (optional)
job: object
```
```json
// example (Interval & Run Commands)
{
  "name": "Made with the API <3 V2",
  "enabled": true,
  "timing": {
      "repeat": true,
      "interval": 200
  },
  "job": {
      "commands": [
        "/say hello from the api",
        "/say this a second command"
    ]
  }
}
```
